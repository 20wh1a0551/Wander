# Wander
A travel website that gives you details about a place when you search for it. It also contains featured destinations, and a blog.
Built using NodeJS, ExpressJS, database integration and uses an API-Key.
To use the website, please generate an API-Key at https://rapidapi.com/trueway/api/trueway-places/ and place it in the index.js file where it says 'YOUR-API-KEY'
